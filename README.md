## ğŸ› ï¸ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone <https://github.com/ShreyashPrateek/portfolio.git>
   cd portfolio-2025
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```
